## 🚀 상세 커리큘럼

###  [1주차] 데이터 모델링의 이해 & SQL 기본기 다지기
**🎯 학습 목표:** SQLD 1과목 이론을 마스터하고, 코딩 테스트에 자주 출현하는 SQL 기초 문법을 체화합니다.

*  **SQLD 이론 (1과목)**
  - [ ] 데이터 모델링의 이해 (엔터티, 속성, 관계, 식별자)
  - [ ] 데이터 모델과 성능 (정규화와 성능, 반정규화, 관계와 조인 성능)

*  **SQL 코딩 테스트 실전**
  - **테마:** 조건에 맞는 데이터 필터링 및 정렬
  - **주요 문법:** `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`, `LIKE`, `IN`, `IS NULL`
  - **추천 문제:** 프로그래머스 SQL 고득점 Kit 중 **'SELECT'** 카테고리 (Lv.1 ~ Lv.2) 5~7문제 풀이

---

###  [2주차] SQL 기본 및 활용 (기본 쿼리 & 집계 함수)
**🎯 학습 목표:** SQLD 2과목 앞부분을 학습하며 데이터 조작 문법을 익히고, 코테의 핵심인 그룹화와 집계 함수를 정복합니다.

*  **SQLD 이론 (2과목 - 1부)**
  - [ ] SQL 기본 (DDL, DML, DCL, TCL)
  - [ ] WHERE절, 함수(단일행/다중행), GROUP BY, HAVING절

*  **SQL 코딩 테스트 실전**
  - **테마:** 데이터 그룹화 및 통계 산출
  - **주요 문법:** `GROUP BY`, `HAVING`, `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`, `CASE WHEN`
  - **추천 문제:** 프로그래머스 SQL 고득점 Kit 중 **'SUM, MAX, MIN'**, **'GROUP BY'** 카테고리 (Lv.1 ~ Lv.3) 5~7문제 풀이

---

### [3주차] SQL 활용 (조인 & 서브쿼리 완벽 이해)
**🎯 학습 목표:** SQLD 합격의 당락을 결정하는 조인과 서브쿼리를 완벽히 이해하고, 다중 테이블 조회 문제를 해결합니다.

*  **SQLD 이론 (2과목 - 2부)**
  - [ ] 조인 (INNER, NATURAL, OUTER, CROSS JOIN)
  - [ ] 서브쿼리 (단일행/다중행, 다중칼럼, 연관 서브쿼리)
  - [ ] 집합 연산자 (UNION, UNION ALL, INTERSECT, MINUS)

*  **SQL 코딩 테스트 실전**
  - **테마:** 다중 테이블 결합 및 다중 조건 필터링
  - **주요 문법:** `INNER JOIN`, `LEFT/RIGHT JOIN`, `서브쿼리(In-line View, 중첩 서브쿼리)`
  - **추천 문제:** 프로그래머스 SQL 고득점 Kit 중 **'JOIN'**, **'IS NULL'** 카테고리 (Lv.2 ~ Lv.4) 5~7문제 풀이

---

###  [4주차] SQL 고급 활용 & 실전 모의고사
**🎯 학습 목표:** 고난도 주제(계층형, 윈도우 함수)를 정리하고, 실전 모의고사를 통해 시험과 코테를 최종 점검합니다.

*  **SQLD 이론 (2과목 - 3부 & 실전 모의)**
  - [ ] 그룹 함수 (ROLLUP, CUBE, GROUPING SETS)
  - [ ] 윈도우 함수 (RANK, DENSE_RANK, ROW_NUMBER, LEAD, LAG)
  - [ ] 인덱스와 옵티마이저 기본 개념
  - [ ] **SQLD 기출문제 및 모의고사 2회분 풀이**

*  **SQL 코딩 테스트 실전**
  - **테마:** 순위 매기기 및 누적합 계산 등 고급 테크닉
  - **주요 문법:** `ROW_NUMBER()`, `DENSE_RANK()`, `DATE_FORMAT`
  - **추천 문제:** 프로그래머스 SQL 고득점 Kit 중 **'String, Date'** 카테고리 및 고난도 복합 문제 (Lv.3 ~ Lv.5) 5문제 풀이
